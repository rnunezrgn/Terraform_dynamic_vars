# Terraform_dynamic_vars

This report describes how Ansible gathers variables from an AAP survey and injects them into Terraform’s variables.tf file to support deployment.


terraform/

	├── main.tf                             # Main terraform configuration file 

	├── variables.tf                        # Define Terraforms variables here
	
	├── terraform.tfvars.json               # This file will be generated by ansible template mode. See line # 15
	
	├── Terraform-Deploy-Ansile-Json.yml    # Ansible playbook to deploy terraform with JSON variables
